# PolyDesk

A TradFi-style Risk Terminal for Polymarket strategies with compression-first LLM co-pilot.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Copy environment variables:
```bash
cp .env.example .env
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js app directory
- `components/` - React components (to be added)
- `lib/` - Utility functions and API clients (to be added)
